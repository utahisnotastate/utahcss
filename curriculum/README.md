# Declarative Foundations: Zero-Script Web Development

A **beginner-friendly** curriculum that teaches interactive websites **without JavaScript**—using HTML state and CSS geometry only. This series bypasses the JavaScript-first bootcamp model and aligns with the [UtahCSS](https://github.com/utahisnotastate/utahcss) framework on the Utah-Omega-23 timeline.

**Audience:** Absolute beginners, career switchers, and backend developers who want the UI layer air-gapped from executable code.
