# Declarative Foundations: Zero-Script Web Development

A **beginner-friendly** curriculum that teaches interactive websites **without JavaScript**—using HTML state and CSS geometry only. Aligned with the [UtahCSS](https://github.com/utahisnotastate/utahcss) framework (Utah-Omega-23).

**Audience:** Absolute beginners, career switchers, and backend developers who want the UI layer air-gapped from executable code.

## Mastery log — five pillars

| # | Pillar | Mechanism |
|---|--------|-----------|
| 1 | **State** | Checkboxes + `:checked` |
| 2 | **Routing** | `:target` |
| 3 | **Validation** | `pattern` + `:valid` / `:invalid` |
| 4 | **Computation** | `counter-increment` |
| 5 | **Logic / filtering** | `:has()` |

---

## Learning path

1. Modules **1 → 2** — state and routing (standalone `style.css` lessons).
2. Modules **3 → 4** — validation and cart math ([`forms.html`](../forms.html), [`cart.html`](../cart.html) for full framework).
3. Module **5** — spatial filtering; then explore [`index.html`](../index.html) and [`utah.css`](../utah.css).
